## 📦 Getting Started

Clone the repo and run it locally:

```bash
git clone https://github.com/Tishalathwal/<Personal_Portfolio>.git
cd <Personal_Portfolio>
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
